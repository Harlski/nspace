# 05 — Guest invite system (QR, URL, shortlink)

## Idea (not yet grilled)

Reusable flow to invite **non–Nimiq Space players** into an activity (first use case: World Cup 1v1):

- Host starts a 1v1 (or future mini-game) and shares **QR** (in-person) or **URL** (remote).
- Friend joins as **guest** instantly; can optionally sign in with existing wallet or create one.
- Host sees status: *Waiting for friend…*, *Invite accepted…*, *Friend is signing in…*, etc.
- Shortlink shape e.g. `nimiq.space/{slug}` mapping to the invite attempt.
- Guest-without-wallet is acceptable — reduces friction for playing with friends.
