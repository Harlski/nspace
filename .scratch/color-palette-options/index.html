Build color picker — design options

Today’s hue ring only gives fully saturated mid-tones. Below are four rectangular full-spectrum layouts (light ↔ dark of every hue) styled for the existing frosted HUD. Pick a direction; interaction details can be grilled next.

Reference · current

Hue ring (fixed S / L)

Circular hue only. Center shows selection; hex popover for escape hatch. No pastel or dark variants without typing hex.

Missing: pale / dark versions of each color

Option A · recommended default

Saturation–Value square + hue strip

Industry standard (Figma / Photoshop). X = white→hue, Y = black→bright. Separate hue rainbow. Fast light/dark for any color.

Option B

Hue–Saturation plane + lightness bar

Full rainbow as a rectangle (hue × how washed-out). Lightness is its own bar — clearer “make it darker / lighter” than folding light into the square.

Lightness
Option C · dock-first

Hue rail + wide SV band

Fits the bottom build dock: thin hue spectrum beside the swatch, taller SV rectangle below. Same math as A, layout tuned for landscape HUD chrome.

Option D · precise

Stacked H / S / L sliders

Three 1D bars. Easiest to nudge lightness alone; denser mentally, less “browse the spectrum.” Good if players often refine rather than explore.
