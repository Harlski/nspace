# reason_628401 — Build dock context height ≤ tool cards

## Why update THE-LARGER-SYSTEM

The Floor SV color picker first shipped taller than the tool cards, stretching the dock and leaving empty space under FLOOR / NO-WALK / SPAWN. Separately, **Use room center** leaked beside the picker because `display: flex` beat `[hidden]` on the spawn-actions row.

## How it serves the larger system

- **Authoring HUD:** Makes the tool-card strip the height authority for the dock body so new context controls stay compact and do not fight the card row.
- **Join Spawn:** Clears custom spawn remains available where guest-entry chrome already lives; the Floor tool strip only needs “click a walkable tile,” not a second center-reset control that crowded the color column.
