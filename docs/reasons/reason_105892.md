# Reason 105892 — `THE-LARGER-SYSTEM` companion files under `docs/reasons/`

**Linked change:** [THE-LARGER-SYSTEM.md](../THE-LARGER-SYSTEM.md) — companion rationale files now live under **`docs/reasons/`** instead of **`docs/`** root.

## Why the system was updated

Rationale files were accumulating next to topical guides in **`docs/`**, which made the top-level index noisier and mixed **time-bound decision provenance** with **evergreen how-to** content. A dedicated **`docs/reasons/`** directory keeps audit-style write-ups grouped and leaves **`docs/`** clearer for operational and feature documentation.

## How this better supports THE-LARGER-SYSTEM

- **Coherence:** One obvious place to look for “why did this principle change?”
- **Evolvability:** Room for more companion files without flattening the rest of `docs/`.
- **Clear direction:** Contributors add new files as `docs/reasons/reason_{unique_6digit_id}.md` only.
