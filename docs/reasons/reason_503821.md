# reason_503821 — Idiomatic in-world UI copy

## Why `THE-LARGER-SYSTEM.md` was updated

We solidified a cross-cutting UX principle: **menus and in-world prompts stay short and idiomatic** so the HUD stays readable in play and does not compete with the world for attention.

## How this supports the larger system

- **Coherence** — One verb phrase per action matches how players scan controls; long explanatory copy in context menus felt like documentation pasted into gameplay.
- **Evolvability** — Product explanations and policy text can live in docs, patch notes, or dedicated help without bloating every interaction surface.
- **Direction** — Future features default to terse labels; deeper explanation is opt-in elsewhere.
