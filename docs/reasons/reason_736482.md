# reason_736482 — Nimiq Pay first-contact tutorial

**Why:** Nimiq Pay first-time players need a sub-minute receive-and-send demo before social Hub
space, without slow on-chain verification blocking the lesson.

mandatory lessons without hiding the layout from operators.
