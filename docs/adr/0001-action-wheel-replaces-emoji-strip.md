# Action Wheel replaces the direct quick-emoji strip

Right-clicking / long-pressing your own avatar used to open a flat quick-emoji strip that
sent an emote in one click. We replaced it with the **Action Wheel**: a radial menu whose
root holds **Sectors** (Emotes, Games, and future features), so emotes now live one level
deeper in an **Emote Wheel**. We accepted that extra click to gain a single, consistent
entry point for all self-actions and room to grow (Games, and more Sectors) without
re-teaching a new gesture each time. A future reader seeing emotes "buried" should know
this was deliberate — not a regression to be flattened back out.
