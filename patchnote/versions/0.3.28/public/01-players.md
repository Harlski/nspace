# Public patch notes — players (`0.3.28`)

**Audience:** people who play or explore Nimiq Space — features, fixes, and feel; not implementation detail.  
**Depth:** short bullets or short paragraphs; avoid file paths and internal names unless they help (e.g. a renamed control).

---

- [NEW] **Build helpers for rooms** — admins can now grant specific wallets permission to build and edit in a room, so it's no longer just the owner. If you're added as a builder, your build tools appear the next time you enter that room.
- [NEW] Admins have a new behind-the-scenes room manager to look inside any room and tidy up its settings (name, visibility, background) without having to walk into it.
- [NEW] Admins can drop **gold, mineable blocks** straight from the build bar — a quick **Gold** toggle next to the rotate buttons — so reward blocks for players to mine are easier to place. Selecting an existing block shows the same toggle, so admins can turn any block gold (or back to normal) without rebuilding it.
