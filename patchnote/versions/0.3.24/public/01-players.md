# Public patch notes — players (`0.3.24`)

**Audience:** people who play or explore Nimiq Space — features, fixes, and feel; not implementation detail.  
**Depth:** short bullets or short paragraphs; avoid file paths and internal names unless they help (e.g. a renamed control).

---

- **[FIX]** **Report Message** (right-click a chat line) — the reported player, wallet, and message appear in a **read-only** box. You only type **why** you are reporting; you cannot change the quoted text. The feedback **Type** picker is hidden for reports.
- **[CHANGE]** The team’s admin tools now include **recent chat from the reported player** when they review your report, so context is preserved even if chat scrolls away in-game.
