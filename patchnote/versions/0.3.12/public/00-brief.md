# Public patch notes — brief (`0.3.12`)

**Audience:** widest reach — social posts, in-app one-liner, “what changed at a glance.”  
**Depth:** minimal; no jargon; link to longer tiers if published.

---

- **[NEW]** After a brief disconnect or room change, **recent room chat** can show again when you land.
- **[NEW]** **Signposts** show a small **document** marker above them so they are easier to notice, especially on touch; the marker **hides behind other blocks** when something is stacked on the tile or blocks the view.
- **[NEW]** Operators can **warn everyone before a server restart** — players see a short countdown banner and a clearer message if the connection drops for maintenance. **GitHub deploys** can send a **60s** warning automatically when a secret is set on the server.
