# Public patch notes — players (`0.3.17`)

**Audience:** people who play or explore Nimiq Space — features, fixes, and feel; not implementation detail.  
**Depth:** short bullets or short paragraphs; avoid file paths and internal names unless they help (e.g. a renamed control).

---

- **[NEW]** **Floor colors** — in build mode, choose **Room → Floor**. Use the **hue ring** (center click for custom hex) to tint tiles: **left-click** places new floor or **recolors** tiles already there, **including under blocks and props**. **Right-click** (desktop) removes extra floor tiles you added.
- **[NEW]** **Cube rotation** — when placing a plain **cube**, use **Rot X / Y / Z** in the dock (or **↺ ↻** / **R** for a quick Y spin) to orient the block before placing. Visual only; footprint stays the same.
- **[FIX]** Painted floor tiles **match the lighting** on blocks and shapes, so the same green (or any color) no longer looks neon-bright on the ground.
- **[FIX]** **Cleaner floor seams** — colored floor grids should look smoother when you zoom and pan.
- **[FIX]** Recoloring floor under a built tile or object works again instead of silently doing nothing.
